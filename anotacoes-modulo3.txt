		GIT E GITHUB

Git: Software que te permite fazer um repositório local.
Github: Site que te permite hospedar seu repositório local.
Repositório local: arquivos do seu computador(git).
Repositório remoto: onde você hospeda seu repositório local.

Atualizar o repositório remoto: "push".


	GITHUB EM VARIOS PC:

. Abra o github;
. Logue na sua conta;
. Clone o repositório para a máquina que não tem;
. Clique em "file", "git" e confirme as credenciais;
. Depois de todas as atualizações no projeto...
. Atualize no github com o push;

	SAINDO SEM DEIXAR RASTROS:

. Feche o visual studio;
. Clique em "repositorio" e então "remove";
. Clique em "file", "options" e deslogue da sua conta;
. Clique em "file", "git", e apague as credenciais;
. Apague o repositório da lixeira.

	DEIXANDO O BODY COM O TAMANHO DA VIEWPORT

<style>
body{
height: 98vh(view height);
}
</style>

	POSICIONANDO IMAGENS NO BACKGROUND

<style>
body{
    background-position: right bottom/right top/right center/left center/center center/ (a lógica é sempre X, depois Y)
}

</style>

	TAMANHO DA IMAGEM NO BACKGROUND

<style>
	body{
	   background-size: contain(exibe sempre a imagem toda)/100%(exibe a imagem adaptável, mas achata a imagem = ruim),
	auto(padrão), cover(cobrir a tela inteira mesmo que tenha que dar uma "picotadinha" na imagem);
}
</style>

	FIXAR IMAGEM NO BACKGROUND(para não rolar junto)

comando: background-attachment(vínculo): fixed(pra fixar dãa)/scroll(padrão);

	SHORTHAND PARA BACKGROUND:

ORDEM: 
1. Cor;
2. Imagem;
3. Posição;
4. Repetição(ou não);
5. Vínculo.

Exemplo: background: black(color) url('imagens/xxxx.jpg')(img) top center(position) no-repeat(repeat) fixed(attachment).

	ALINHAMENTO VERTICAL DE CAIXA

criar um conteúdo dentro de um container => container >> conteúdo:

<section id="container">
	<article id="conteudo">

	</article>
</section>

o CONTAINER, vai ter posição relativa(position: relative);
o CONTEUDO, vai ter posicionamento absoluto(position: absolute);

Quando você coloca o posicionamento absoluto, você consegue mexer nos seguintes parâmetros:
-top
-left

Utilizar o transform: translate() -> para mexer na caixa, podendo usar píxel, porcentagem...

	ANATOMIA DE TABELAS GRANDES

     Table
	Thead
	   tr(linha), td(dado), th(título)
	Tbody
	   tr, td, th
	Tfoot
	   tr, td, th

	SEMÂNTICA DOS ESCOPOS

<Caption>Coloca um título acima da tabela</caption>
<th scope="col">Quer dizer que esse th está no escopo de título de COLUNA, ou seja, os dados virão ABAIXO desse título.
<th scope="row">*** Título de LINHA, ou seja, os dados virão AO LADO desse título.

	MESCLAGEM DE CELULA

colspan="2" dentro da tag <td>
exemplo: 
<table>
  <tr>
      <td colspan="2">
	
colspan para coluna
rowspan para linha


	SCOPE

Colocar cada escopo para cada <th> da tabela!
scope=row: linha de escopo, onde os dados estarão ao lado.
scope=col: coluna de escopo, onde os dados estarão a baixo.

colgroup: quando há mesclagem de colunas.
rowgroup: mesclagem de linhas.
